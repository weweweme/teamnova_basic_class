# 기획안: 쿠키런 스타일 사이드스크롤

## 개요
- 일자로 계속 진행되는 사이드스크롤
- (A) | 형태의 캐릭터가 걸어가면서 이벤트 발생
- 몬스터 출현 시 턴제 전투
- 랜덤 이벤트 (자원 발견, 함정, NPC 등)

## 화면 구성
```
═══ 표류자들 ═══ 3일차 낮 ═══ 거리: 1.2km ═══

                         /\
  (A) (B) (C)           /XX\      T   .  o
   |   |   |           /XXXX\    TTT  . ooo
──/────/────/──────────────────────────────────

  [A]철수 HP████░░  [B]영희 HP██████  [C]민수 HP████████
──────────────────────────────────────────────
  >> 앞에 괴물이 나타났다!
  1:전투  2:우회(피로+5)  3:도망(식량-3)
```

## 전투 화면
```
═══ 전투! ═══

  (A)            XX
   |      ⚔     XX
──/────────────/──────

  김철수 HP████████░░     괴물 HP██████░░░░

  > 김철수의 차례
  1: 공격    2: 방어    3: 도망
```

## 스레드 활용
- 정착민 스레드: 배고픔/피로 실시간 증가
- 낮밤 스레드: 밤엔 몬스터 더 강하고 자주 출현
- 이벤트 스레드: 일정 간격 랜덤 이벤트

## 보류 사유
- 스레드가 시각적으로 잘 드러나지 않을 수 있음
